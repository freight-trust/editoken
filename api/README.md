# API 

> API for Interacting with $EDI Token

## Setup

- valid Etherscan.io API Key
- Infura API Key


## Overview

* Exchanges 
  - Buying
  - Selling
* Reporting
  - CMC
  - Coingecko
* Staking
  - Current Status 
  - Epoch
  
## Examples

```bash
$ npm install -g @fr8/token
$ edi-ctl --help
```
